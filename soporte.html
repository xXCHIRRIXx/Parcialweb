Aquí encontrarás toda la información necesaria para hacer tu viaje más fácil y cómodo.

¿Cómo obtener el Check-In?

Para obtener tu Check-In, sigue estos pasos:

  1. Visita nuestra página web o utiliza nuestra aplicación móvil.
  2. Ingresa tus datos de vuelo.
  3. Selecciona tu asiento.
  4. Confirma tus datos y genera tu tarjeta de embarque.

Recuerda que el Check-In se puede realizar 24 horas antes del vuelo.

Requisitos de Viaje

Antes de viajar, asegúrate de tener en cuenta los siguientes requisitos:

  • Documento de identidad válido.
  • Pasaporte (si viajas internacionalmente).
  • Visas requeridas para tu destino.
  • Comprobantes de vacunación, si es necesario.

Consulta con tu aerolínea los requisitos específicos para tu vuelo.
